What is depicture?

depicture is a game you play online with your friends, with gameplay similar to Scrawl, Sketchy Stories, Drawphone, Drawception, and Telestrations.

The project is open-source - in fact, it's dedicated to the public domain, so you are free to do whatever the heck you want with the code! Find it here!

Updates

What's new

  • Fixed pen getting stuck on drawing when leaving the canvas
  • Added quick restart game button (only host sees it)
  • Player names are attached to their work in the end screen
  • Restrictions on text inputs
  • Input for prompt provider
  • Turn ticker during the game to see how far along you are
  • Reveal end stories piece by piece

What's missing
