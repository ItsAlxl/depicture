What is depicture?

depicture is a game you play online with your friends, with gameplay similar to Scrawl, Sketchy Stories, Drawphone, Drawception, and Telestrations.

The project is open-source - in fact, depicture is dedicated to the public domain, so you are free to do whatever the h*ck you want with the code! Find it here!

Updates

What's new

  • Basic mobile support
  • Save games (with matching CSS!)
  • Deal with drop-in/drop-out gracefully
  • Short join codes
  • Styling
  • Some invalid action feedback
  • Host can specify a stage limit
  • Player names of who you're waiting on
  • Shuffle player order between rounds
  • Reveal end stories piece by piece
  • Turn ticker during the game to see how far along you are
  • Input for prompt provider
  • Restrictions on text inputs
  • Player names are attached to their work in the end screen
  • Added quick restart game button (only host sees it)
  • Fixed pen getting stuck on drawing when leaving the canvas

What's missing

  • Various host options (player limit, disable inter-round shuffling, public/private game, etc.)
  • Allow spectators
  • Graceful host transfer
  • Feedback for invalid actions
  • Better looks
  • Better mobile support
  • Easier way to add custom prompt lists
  • Players can ready/unready in lobby
  • Public game list
  • A smaller pen size
  • A yellow pen
